# CreationEngineAI


## Requirements

Get OpenAI API key [here](https://openai.com/api/).

## Running Locally

1. Clone repo

```bash
git clone https://github.com/Agora-X/CreationEngine
```

2. Install dependencies

```bash
npm i
```

3. Run app

```bash
npm run dev
```
